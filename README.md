# Ruboty::Postgres

store memory of [ruboty](https://github.com/r7kamura/ruboty) in postgresql

## ENV

- `POSTGRES_URL`
- `POSTGRES_USER`
- `POSTGRES_PASSWORD`
- `POSTGRES_DATABASE`
- `POSTGRES_NAMESPACE`
